# Naukri Automation Framework

A comprehensive, production-ready Selenium automation framework built using Java, Selenium WebDriver, TestNG, Maven, and modern automation testing practices.

---

# 🚀 Features

## Core Framework

- Enhanced Page Object Model (POM)
- Thread-safe WebDriver management using ThreadLocal
- Cross-browser execution support
- Parallel execution using TestNG
- Custom utilities for waits, screenshots, and JavaScript execution
- Comprehensive logging using Log4j2
- Detailed reporting using ExtentReports
- Retry mechanism for failed test cases
- CI/CD integration support
- Headless browser execution support

---

# 🧪 Supported Testing Types

- Functional Testing
- Regression Testing
- Smoke Testing
- API Testing using REST Assured
- Database Testing using JDBC
- Performance Testing using JMeter
- Security Testing using OWASP ZAP
- Accessibility Testing
- Mobile Testing using Appium
